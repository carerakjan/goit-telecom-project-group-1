Інструкція Користувача

Цей додаток дозволяє аналізувати клієнтські дані та визначати ймовірність припинення використання послуг.

1. Вимоги до Системи
Перед початком роботи переконайтеся, що у вас встановлено Docker та Docker Compose для запуску проекту в контейнеризованому середовищі.

2. Запуск Системи
Завантаження проекту: Завантажте архів з кодом проекту або клонуйте репозиторій за допомогою Git.
Запуск Docker Compose:
•	Відкрийте термінал у директорії проекту.
•	Виконайте команду docker-compose up для створення та запуску необхідних Docker контейнерів.
•	Ця команда автоматично сконфігурує та запустить всі необхідні сервіси.

3. Використання Системи
Доступ до Додатку:
•	Перейдіть за посиланням до хмарного додатку через веб-браузер.
•	Авторизуйтеся, якщо це необхідно.

4. Головна Сторінка
Інструкція Користувача: 
•	Доступ до інструкції здійснюється натисканням на кнопку 'Інструкція Користувача'.
Початок Аналізу: 
•	Для прогнозування відтоку клієнтів натисніть 'Почати Аналіз', для переходу до сторінки 'Прогнозування клієнтів'.

5. Сторінка Прогнозування Клієнтів
Одиночне Прогнозування: Вибір цієї опції дозволяє прогнозувати відтік для одного клієнта.
•	Для вибору одиночного прогнозування клієнтів натисніть на кнопку 'Одиночне Прогнозування'
Множинне Прогнозування: Вибір цієї опції дозволяє прогнозувати відтік для декількох клієнтів одночасно.
•	Для вибору множинного прграмування клієнтів натисніть на кнопку 'Множинне Прогнозування'

6. Сторінка Одиночне Прогнозування
Введення Даних: 
•	Введіть дані клієнта у відповідні поля та натисніть 'Прогнозувати' для отримання результатів.
Результати: 
•	Результати будуть відображені як висока або низька ймовірність відтоку та у вигляді відсотків в відповідних полях.
Збереження Результатів: 
•	Для збереження результатів натисніть кнопку 'Зберегти результат'. 

7. Сторінка Прогнозування Множинне
Завантаження Датасету:
•	Використовуйте поле 'Drag and drop' для завантаження датасету методом 'drag and drop' або натисніть кнопку 'Browse file' для завантаження датасету з локального середовища
Формат Датасету: 
•	Датасет повинен бути у форматі .csv
•	Датасет повинен включати в себе наступні поля:
         •	is_tv_subscriber: інформація про підписку на телебачення.
         •	is_movie_package_subscriber: інформація про підписку на пакети фільмів.
         •	subscription_age: тривалість підписки.
         •	remaining_contract: термін, що залишився до кінця контракту.
         •	download_avg: середня швидкість завантаження.
         •	upload_avg: середня швидкість вивантаження.
         •	download_over_limit: перевищення ліміту завантаження.
Аналіз та Результати: 
•	Натисніть 'Прогнозувати' для обробки даних та отримання результатів.
Результати: 
•	Результати будуть відображені як висока або низька ймовірність відтоку та у вигляді відсотків в відповідних полях.
Збереження Результатів: 
•	Для збереження результатів натисніть кнопку 'Зберегти результат'.

8. Перегляд Результатів
Відображення Результатів: 
•	Результати аналізу відобразяться на сторінці з показником відтоку.
Візуалізація: 
•	Результати аналізу будуть представлені на сторінці у вигляді графіків.

9. Обслуговування та Підтримка
Контактна Інформація: 
•	Для отримання допомоги або в разі виникнення проблем з додатком, звертайтеся до служби підтримки через зазначені контактні дані email або телефон служби підтримки.
